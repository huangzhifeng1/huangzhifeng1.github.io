下载与安装

官方下载:https://git-scm.com/download/linux

1.官方网站下载安装包

2.将安装包放入虚拟机指定位置

3.解压安装包

tar -zxvf git-2.34.1.tar.gz

`

4.安装编译环境

yum install curl-devel expat-devel gettext-devel openssl-devel zlib-devel gcc perl-ExtUtils-MakeMaker

`

`

注意:如果之前有下载过git可以使用yum remove git命令进行卸载！

5.编译源码

make prefix=/usr/local/git

`

6.安装git

  make prefix=/usr/local/git

安装完会出现以下目录

`

7.配置环境变量

vim /etc/profile

`

最后刷新环境变量！

source /etc/profile
git --version `查看版本`

`

这样git就安装完成啦！！！！

配置git相关信息

1.先在gitee上创建一个项目

`

2.将相关信息与git绑定

git config --global user.name“黄志锋"
git config  --global user.email "9844272+huang-zhifeng1@user.noreplygitee.com

`

3.推送test文件测试是否推送成功

git init `初始化项目`
git add . `将当前目录下的文件添加到仓库（缓冲区）`
git commit -m "init project" `提交到本地仓库`
git push -u origin master `推送到远程仓库`

`

查看gitee是否有成功推送的信息！！
